# DORM MATCHING API DOCUMENT

### `[POST] /api/get_all_identities`
> get all identities from the given student file

Request
```
[
    [col1, col2, col3,...],
    [atr1, atr2, atr3,...].
    [atr1, atr2, atr3,...],
    ...
]
```

Response 200 OK Success
```
[
    identity1,
    identity2,
    identity3,
    identity4,
    ...
]
```
### `[POST] /api/match`
> return all matching results

Request
```
{
    "priority":[
        [identify1, identify2],
        [identify3],
        ...
    ],
    "student":[
        [col1, col2, col3,...],
        [atr1, atr2, atr3,...].
        [atr1, atr2, atr3,...],
    ],
    "beds":[
        [dormName, Room, Bed, Student_ID, is_disability,...],
        [atr1, atr2, atr3,...],
        [atr1, atr2, atr3,...],
    ]
}
```
